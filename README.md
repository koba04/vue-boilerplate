# My Vue.js Boilerplate

* vue.js
* coffeescript
* browserify
  * beefy (auto build & livereload)
  * partialify
  * coffeeify
  * stylify
* insert-css

## setup
```
npm install
```

## start develop
* livereload and autobuild
  * not created bundle.js
```
npm run dev
```

## build
* create bundle.js
```
npm run build
```
